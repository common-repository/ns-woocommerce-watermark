== Description ==

**Apply your watermark on your WooCommerce products image!**

This plugin work perfectly with new and existing WooCommerce product image. 
You can activate this plugin without overwrite the existing images. 
Choose watermark png, jpg or gif from your local folder or from media library. 
You can enable or disable watermark on the images without any problems.

> You can choose if Watermark can be seen by unregistered users only.
> Plus you can choose if your registered/unregistered users can or cannot download your photos ("save as" is ineffective and original URL is obscured) (NEW FEATURES!).

https://www.youtube.com/embed/ezHKXvcbP1A

== Installation ==

1. Upload the entire 'woocommerce-watermark' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS Woo watermark Options'


== Frequently Asked Questions ==

= Is required WooCommerce to use this plugin? =

Yes WooCommerce is required to use this plugin.

= This plugin create a new image? =

No, the image are only viewed.

= This plugin overwrite my image? =

No, the image are not overwrite.

= Code is required to use this plugin? =

No code require to use this plugin.

= Plugin doesn't work? =

Check if gd library are install and allow_url_fopen is on.
If it is off, contact your admin server.
